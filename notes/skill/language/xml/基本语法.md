# 语法规则
xml必须要有根节点
即所有节点的父节点

<?xml version="1.0" encoding="UTF-8"?>
这句话可有可无


所有xml必须是成对的标签

标签大小写敏感

根节点不一定非要用boot标签，可以用其他标签，xml的标签是可以自定义的，但是要有意义，比如package

标签不能交叉

<!-- 这是注释 -->

遇到到特殊字符时，必须使用实体引用
比如<要写成&lt;，>要写成&gt;，&要写成&amp;，"要写成&quot;，'要写成&apos;
就只有这五个特殊字符需要使用实体引用，其他的字符都不需要
