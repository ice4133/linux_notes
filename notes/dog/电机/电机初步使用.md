# 快速使用
本文仅在ubuntu22.04上测试，windows自行测试，编译所需的环境自己配，只需要c++环境即可

首先拿到官方给的压缩包

进入~/SDK/unittree/Linux平台电机使用例程(包含SDK)$ 

// 这个路径是我自己的路径，大家根据自己的路径进入
// CMakeLists.txt  example  include  lib  LICENSE  motor_tools  README.md
// 该目录下有这些文件


```bash
# 在刚刚给的路径下，打开终端
mkdir build
cd build
cmake ..
make

```

接好线（找人教）

```bash
# 仍然在build目录下，打开终端
sudo ./motorctrl
# 此时电机转动

ctrl+c

# 停止
sudo ./stop

ctrl+c

```

#include "serialPort/SerialPort.h"
#include <unistd.h>

int main() {

  SerialPort  serial("/dev/ttyUSB0");
  MotorCmd    cmd;
  MotorData   data;

  while(true) {
    cmd.motorType = MotorType::GO_M8010_6;
    cmd.id    = 0;
    cmd.mode  = 1;
    cmd.K_P   = 0.0;
    cmd.K_W   = 0.05;
    cmd.Pos   = 0.0;
    cmd.W     = 6.28*6.33;
    cmd.T     = 0.0;
    serial.sendRecv(&cmd,&data);

    /*如果只是进行上位机调试，那么我们就只看这几个数据

    mode 默认为1,因为这是foc模式；如果设置为0,就是关闭电机；2是标定模式。故mode为1

    接下来几个数据，就是核心控制了

    具体直接看文档
