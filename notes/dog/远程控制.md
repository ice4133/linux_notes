```bash
# 下载插件
remote-ssh


# 点击“打开SSH配置文件” ，就是一个齿轮

# 输入
Host dog # 只有这个dog是随便取的
    HostName 172.20.10.2
    User neuq


# 这到底是在做什么？
# 你可以把这个 config 文件 想象成你电脑里的**“拨号快捷方式”**：

# Host (昵称)：就像联系人姓名（比如“老王”）。

# HostName (IP)：就像对方的电话号码。

# User (用户名)：就像你要找的人在那个家里的身份。

# 以前你需要每次在终端输入长长的 ssh ubuntu@192.168.1.50，现在你只需要在 VS Code 里点一下“老王”，它就会根据这个配置文件，自动帮你把长指令拼好并发送出去。




# 然后点击新建终端连接

# 或者直接
ssh neuq@172.20.10.2

# ctrl + D 来退出 或者输入 exit 来退出

# 必须在minipc还在的时候可以这样，如果掉电了，就必须强制退出了
```
