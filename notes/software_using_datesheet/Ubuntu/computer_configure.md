# 左侧任务栏取消
开启“自动隐藏”（推荐）
如果你希望在看代码或文档时任务栏不占位，但偶尔还需要用它，这是最平衡的选择。

+ 打开 设置 (Settings)。

+ 在左侧栏找到 外观 (Appearance)。

+ 找到 Dock 这一节。

+ 开启 自动隐藏 Dock (Auto-hide the Dock)。

+ 开启后，当有窗口覆盖到左侧区域时，任务栏会自动消失。你只需要将鼠标快速移动到屏幕最左侧，它就会再次浮现。
