# 类比

|维度|手机操作系统 (如 Android)|机器人操作系统 (ROS)|
|---|---|---|
|底层内核|基于 Linux 内核|基于 Linux 内核|
|硬件抽象|摄像头、GPS、加速度计的驱动|雷达、IMU、电机驱动、机械臂接口|
|软件单元|应用程序（App）|节点（Node）|
|通信机制|Intent、Broadcast Receiver|Publisher-Subscriber、Service、Action|
|功能扩展|应用商店|GitHub、ROS Index|
|用户界面|图形用户界面（GUI）|命令行界面（CLI）、RViz、rqt|
|开发语言|Java、Kotlin、C++|Python、C++|
