针对你的 C++ 编写的底层控制程序。   
命令： top -p <你的进程PID> 或 ps -eLo pid,tid,class,rtprio,stat,comm
看什么： 重点看 PR (优先级)。对于机器狗底层，你可能需要设置 实时调度策略（FIFO）。



ROS 2 数据流监控
频率检测： ros2 topic hz /pointcloud 或 /image_raw

逻辑标准： 如果你设定的频率是 10Hz，但实际只有 7Hz，说明性能已经跟不上了。

带宽检测： ros2 topic bw /image_raw

逻辑标准： 相机如果占用超过 500MB/s 的带宽，DDS 序列化会消耗极大的 CPU
