# ubuntu配置

+ 到github上面下载

+ 配置环境变量
打开终端
vim ~/.bashrc

```
# MuJoCo 配置
export MUJOCO_PATH="/home/ice/Apps/mujoco-3.5.0-linux-x86_64/mujoco-3.5.0"
export PATH="$MUJOCO_PATH/bin:$PATH"
export LD_LIBRARY_PATH="$MUJOCO_PATH/bin:$LD_LIBRARY_PATH"
```

+ 打开并使用

找到与bin文件夹同级的model/humanoid/humanoid.xml

```
simulate /home/ice/Apps/mujoco-3.5.0-linux-x86_64/mujoco-3.5.0/model/humanoid/humanoid.xml
```
