# 针对8086
sudo apt update
sudo apt install dosbox
sudo apt install nasm


下载插件MASM/TASM


然后对这个插件进行配置
选择仿真器 (Emulator)：

找到 MASM-TASM: Emulator。

建议选择：dosbox（这是 Linux 下最稳定的选择）。

选择编译器 (Assembler)：

找到 MASM-TASM: Assembler。

建议选择：MASM（最经典的微软汇编格式，教程也最多）。

运行后的动作 (Action)：

建议勾选 MASM-TASM: Save All Files Before Run。这样你每次运行前不需要手动 Ctrl+S。
