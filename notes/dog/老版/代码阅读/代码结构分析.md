dog_ws/src/
├── robot/                          # 主控制包（核心功能）
│   ├── robot/                      # Python模块目录
│   ├── launch/                     # 启动文件
│   ├── resource/sdk_lib/           # Unitree SDK库文件
│   ├── package.xml                 # ROS2元数据
│   └── setup.py                    # Python入口点配置
│
└── unitree_legged_msgs/            # 消息定义包
    └── msg/MotorState.msg          # 自定义电机状态消息

硬件驱动层文件结构说明：
```bash
unitree_go_1.py # 主控制节点 - 与Unitree SDK交互，驱动12个关节电机和4个轮子


motor_init.py  
motor_init2.py # 电机初始化和参数配置


imu_process_node.py # IMU传感器节点 - 读取IMU数据，进行低通滤波处理，发布滤波后的IMU信息



# 导入 unitree_actuator_sdk 库进行硬件控制
# 低通滤波器平滑传感器噪声
# 关节电机初始角度：[2.17, 0.12, 5.02, 0.42, 3.38, 4.77, 3.09, 4.45, 0, 0, 0, 0]
```


控制决策层文件结构说明：
```bash
decision_control.py # 核心决策引擎 - 包含PID控制器、状态机、路径规划

manual.py # 手动遥控模式（键盘控制） 遥控器没写

auto_obstacle.py # 自动避障逻辑



# PID控制器: 用于平滑控制电机转速和方向
# 支持积分分离（误差大时不积分）
# 支持积分限幅（防止积分饱和）
# 过程数据处理: 从IMU获取姿态，处理偏航角/横滚角/俯仰角

```


通信层文件结构说明：
```bash
upper_comms_node.py # 与上位机通信（ROS Topic订阅-发布），传输命令/数据

serial_node.py   # 串口通信（物理接口）
```

数据结构：
```python
{
    'imu': {},           # IMU数据
    'label': '',         # 任务标签
    'distance': 0.0,     # 距离信息
    'x_offset': 0.0,     # 位置偏差
    'is_qr': False,      # 二维码识别标志
    'qr_content': ''     # 二维码内容
}
```

消息定义
```bash
MotorState.msg:
├── mode              # 电机工作模式
├── tau_est           # 估计力矩
├── q                 # 关节位置
├── dq                # 关节速度 (rad/s)
├── kp / kd           # PID参数

```
